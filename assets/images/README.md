# Images and Visual Assets

Store diagrams, charts, and visual documentation here.

## Structure

- `architecture/`: System architecture diagrams
- `charts/`: Data visualizations and charts  
- `screenshots/`: Application screenshots

## Supported Formats

- SVG (preferred for diagrams)
- PNG (for screenshots)
- JPG (for photos)

## Tools

Recommended tools for creating diagrams:
- Draw.io
- Lucidchart
- Mermaid (for code-based diagrams)
